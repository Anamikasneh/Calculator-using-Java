# Simple Calculator Project using Java

This is a basic calculator project implemented in Java. The project provides a simple graphical user interface (GUI) for performing arithmetic calculations, including addition, subtraction, multiplication, and division.

## Features

- User-friendly GUI with buttons for digits and arithmetic operations.
- Supports basic arithmetic operations: addition, subtraction, multiplication, and division.
- Handles user input from both buttons and keyboard.
- Error handling for common scenarios, such as division by zero and invalid input.
- Clear and reset functionalities to start new calculations.
- (Optional) Memory functions for storing and recalling values.
- Thoroughly tested for accurate calculations and responsive behavior.

## How to Run

1. Clone this repository to your local machine using:


2. Make sure you have Java Development Kit (JDK) installed on your machine.

3. Open the project in your preferred Java Integrated Development Environment (IDE).

4. Navigate to the `src` folder and find the `CalculatorApp.java` file.

5. Run the `CalculatorApp.java` file to launch the calculator application.

6. Use the GUI buttons or keyboard input to perform calculations.
